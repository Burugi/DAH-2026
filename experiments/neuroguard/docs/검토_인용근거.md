1. **B5(A18·B5 본문)** — "휴리스틱이 PPO ~5배"를 본문서 그대로 쓰면 과장. 본문 B5/A1는 "휴리스틱이 PPO를 ~5배 앞섬"으로 기재됨 → **"약 4–5배"** + **"DRL 아닌 결정론적 휴리스틱"** 명시 필요.
2. **RQ-170(A14·D1)** — 정리본 A14는 "RQ-170=이란 주장+plausible"로 비교적 양호하나, 본문 다른 곳서 확정 사실처럼 쓰지 않도록 최종 점검.
3. **VLM 인젝션 "594/4종"** — `인용논문_정리.md`·`확장아이디어_리서치.md`에 등장. **수치 근거 불일치 → 삭제/재확인.**
