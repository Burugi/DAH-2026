## 6. 최종 종합 (한 장)

```
공격: 점령형(다 막힘) · 가용성 · 탐지 · 연결성(★유일 약점)
방어: 예측 → 가용성인지 → MPC → 코디네이터(0.74) → ★통합 방어(평균 0.66/worst 0.51)
학습: 명시 설계 > 모방 > from-scratch RL
천장: 연결성/센싱 → 해답 = 2-연결성 + 분산 자율
```

> **최종 권장 = 통합 방어(분산 자율 코디네이터 + de-jam + relay + 가용성 보존).** 본선 1순위 투자 = 빠른 탐지 +
> 넓은 커버리지(연결성·형상 이중화) + 협조 + 가용성 보존.
