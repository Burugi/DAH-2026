## 1. 치명적 교차결함 (우선순위)

### C1. 탐지 ↔ 은밀 공격 모순 — **가장 중요** (방어·에이전트 검토가 독립적으로 동일 지적)
- **문제:** B20/C16의 L1은 "불변식·임계 위반으로 미지 공격도 *반드시* 잡는다"고 주장. 그러나 **A19(Raven)·A3-slow(점진 드리프트)·A21(VLM)** 은 *정상 분포·물리 일관성·센서 노이즈 한도 안에 머물도록 설계된* 공격 → L1·B1·B2·B15를 의도적으로 회피. 즉 우리 카탈로그 안에 우리 탐지 프레임의 반례가 있음.
- **수정:** L1 "반드시 깨진다" → **"많은 공격은 불변식을 깨지만, 노이즈/일관성 한도 내 은밀 공격은 L1 단독으로 못 잡는다"** 로 정직하게. 이를 **L2 교차모달(다중센서·VLM 검증) + 합의 기반 교정(B5 Whistle·B9 BFT·SwarmRaft) + L4 적응**의 다층으로 메우는 서사로 전환. → 약점이 아니라 *"단일 탐지의 한계를 알고 다층 설계"* 라는 견고성 가점 요인.

### C2. A3 ↔ B3 결합 취약점 (웹 실증)
- **문제:** MAVLink 2.0 서명의 anti-replay 타임스탬프가 **GPS 시각 스푸핑으로 우회**됨(RTC 없는 기체는 GPS lock 시 서명 타임스탬프 갱신 → 키 없이 재전송 성립). 근거: ArduPilot Issue #13860, IEEE 2024 "Timestamp Manipulation-Based GPS Spoofing on MAVLink 2.0". 문서는 A3·B3를 별개로 다룸.
- **수정:** ① 공격에선 **A3+A2 결합(시각 스푸핑→서명 우회 재전송)을 강점으로 무기화**, ② 방어 B3엔 **단조 시퀀스 카운터·독립 RTC·GPS시각 비의존 anti-replay** 보강 명시. → 공격·방어 양쪽에서 깊이 가점.

### C3. 매핑 누락·오매핑 (도메인 심사위원 즉시 인지)
| 유형 | 항목 | 조치 |
|------|------|------|
| 누락 | A5 ROS2 → SROS2 직접방어 부재 | B11(제로트러스트)·SROS2 방어 추가 |
| 누락 | A16 ISR 탈취 → 기밀성 방어 부재 | 암호화·DLP(B17 연계) 추가 |
| 누락 | A11/A21 방어AI 공격 → 매핑 약함 | **B14 적대적 견고성**과 명시 매핑 |
| 누락 | A12 사이드채널·A15 내부자 | "물리/인사 보안 영역(대회 범위 외)" 명기 또는 제외 |
| 메타 | A13 시간동기 → B3/B16 자체를 깸 | A13을 B2/B3 한계 절에 연결 |
| 오매핑 | B12↔A7(재밍엔 읽을 신호 없음), B18↔A7(재밍≠레이트리밋), B4↔A11(존차단≠모델공격), B9↔A3(BFT는 물리신호 못막음) | 매핑표 정정 |

### C4. 가용성 자해 리스크 (×가용성 직격)
- **B10 MTD:** 위성 고지연 환경에서 주파수/주소 도약이 green 트래픽을 끊어 가용성 직접 타격 → **순효과 음수 가능**. CC3엔 정찰단계가 없어 MTD 주효과도 미발현. → 본선 한정·조건부로 축소.
- **B1→B7 사슬:** "탐지=가용성 무손실"은 탐지 자체만 참. 오탐이 B7/B19 대응 사다리를 발동 → 자해. **가용성은 상류 탐지 정확도에 종속**임을 인정하고 오탐 억제(green 모델)를 전제로 명시.
- **A6·A7:** 완전 무력화로 흐르면 공격자도 ×가용성 손해 → "미세 왜곡/간헐"의 스텔스 원칙을 A6·A7에도 적용.

### C5. 과장 표현 (완화 시 신뢰도↑)
- **GNN(C8):** "크기 무관 zero-shot 성능 우위" → 근거인 Cybermonic은 **CC4에서 −193으로 휴리스틱(−118)에 졌음**. "토폴로지 변화에 강함(이식성)"으로 완화, 성능 우위 주장 삭제.
- **B20 L1 "반드시"**(위 C1) · **Canary "5배"** → **"약 4~5배"**.
- **C13 월드모델·C9 MARL:** "주력" 오인 금지 → 대안/신중으로 명확히.

## 3. 강점 (그대로 살릴 것)

- **③ 보상·하이브리드가 1차 검증 평결과 정합** — R_red 곱셈→가중합 정정 반영(C2), LLM 실시간 배제·오프라인(C3), 환경 직접구현 금지·CC 채택(C15). *공학적 정확성*이 혁신성의 근거.
- **B5 Canary/Whistle** — CC3 우승 실증·sim 직접구현·경량. 가장 강한 방어.
- **B7+B19 가용성 보존 차등대응** — ×가용성 채점을 학습에 내재화.
- **B15 물리 불변식 잔차** — signature-free 탐지의 *진짜* 근거(미지공격 일부 흡수).
- **A1 군집 웜** — 유일 sim 실현 + CC3 벤치마크 권위 + 가용성 유지형.
- **최강 인용:** FFI RL 사고대응(2410.21407, 보상 −10/−1/+50 채점식 일치·실 UGV) · Incalmo(2501.16466, 37/40 원문 일치).
