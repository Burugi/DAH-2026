## [토글] 도메인 내용 정리

**UAV 시스템 구성 (공격면의 토대)**

| 구성요소 | 설명 | 대표 |
|----------|------|------|
| 비행제어컴퓨터(FCU) | 자세·비행 제어 HW | Pixhawk |
| 오토파일럿 SW | 비행제어 펌웨어/스택 | PX4, ArduPilot |
| 지상통제(GCS) | 임무계획·조종·모니터링 | QGroundControl, Mission Planner |
| 통신링크(C2) | 기체↔GCS 데이터 | RF, 셀룰러, SATCOM(위성) |
| 통신 프로토콜 | 명령/텔레메트리 | MAVLink |
| 항법센서 | 위치·자세 | GNSS(GPS), IMU, 기압계, RTK |
| 페이로드 | 임무 장비 | EO/IR 카메라, SAR, 무장 |
| 컴패니언 컴퓨터 | AI·영상·자율 | Jetson, RPi + ROS2 |
| 임베디드 기반 | RTOS·펌웨어·OTA | NuttX, 부트로더 |

> 🔑 임베디드 강점이 꽂히는 지점 = FCU·오토파일럿·RTOS·펌웨어·센서. UGV는 LiDAR·카메라·SLAM·ROS2 중심 → **센서 스푸핑 위협이 UAV보다 큼**.

**통신·프로토콜 보안 약점**

| 대상 | 약점 |
|------|------|
| MAVLink 1.0 | 인증·암호화 없음(평문) → 명령 위조·주입·스니핑·재전송 |
| MAVLink 2.0 | 메시지 서명(secret-prefix MAC) = 인증·무결성만, **암호화 없음·기본 OFF** |
| SATCOM(위성망) | 높은 지연·간헐 단절·대역폭 제한 → 재밍·재전송·MITM. *대회 환경의 정체성* |
| RF | 재밍(DoS), 미코닝(재방송 기만) |
| ROS2/DDS | SROS2 미적용 시 디스커버리 악용·토픽 인젝션 |

**항법(GNSS) 보안**

| 구분 | 내용 |
|------|------|
| 재밍(Jamming) | 신호 교란 → 측위 불능(DoS) |
| 스푸핑(Spoofing) | 가짜 신호로 위치·시간 기만 → 경로 이탈·탈취 (더 은밀·위험) |
| 민간 신호 | 비암호화 → 취약 / 군용 M-code·P(Y)는 암호화 |
| 방어 | RAIM, 다중 GNSS 교차검증, INS-GNSS 일관성, 안테나 어레이, 신호파워 모니터링 |

**공격면 종합 (계층별)**

| 계층 | 대표 위협 |
|------|-----------|
| 공급망(펌웨어·부품) | 펌웨어 트로이목마, 악성 OTA |
| 통신링크 | 명령 주입·위조, 재밍, MITM, 재전송 |
| 항법(GNSS) | GPS 스푸핑/재밍 |
| 센서 | LiDAR/카메라/IMU 스푸핑 |
| SW/펌웨어 | 익스플로잇, OTA 변조 |
| GCS | 일반 IT 침투 → C2 장악 |
| 컴패니언/ROS2 | DDS 인젝션, 노드 탈취 |
| AI/ML | 적대적 예제, 모델 포이즈닝 |

> STRIDE: Spoofing(GPS/MAVLink)·Tampering(펌웨어/명령)·Info-leak(스니핑)·DoS(재밍)·Elevation(권한상승)

**방산 도메인 특수성 → 설계 원칙**

| 특수성 | 설계 반영 |
|--------|-----------|
| 안전성·가용성 최우선 | `×가용성` 채점 직결 — 과잉 방어=자해 |
| 실시간성(ms, RTOS) | 에이전트 저지연·경량 |
| 자원 제약(임베디드) | 경량 탐지·휴리스틱 |
| 공급망 신뢰 | 출처 검증(공급망 공격 실질 위협) |
| 물리-사이버 융합(CPS) | 사이버 공격→물리 결과(추락·충돌) |
| 적대적 환경(EW·단절) | 자율성·강인성 |

**본선 채점식 & 핵심 인사이트**

| 항목 | 내용 |
|------|------|
| 산식 | `total = (attack + defense) × availability` |

**우리 팀 자산 매핑**

| 팀 자산 | 대회 의미 |
|---------|-----------|
| AI 석사 ×2 | ③ 에이전트(RL/LLM), ② 방어 ML |
| 임베디드 ×1 | ①② UAV/UGV 공격면·현실성 |

> 메시지: "AI 에이전트를 실제 무기체계 도메인에 적용하는 팀" = 순수 보안팀·순수 AI팀이 못 갖는 교집합.

**CAGE 벤치마크 교훈 5가지**

| # | 교훈 |
|---|------|
| 1 | 휴리스틱+RL 하이브리드가 순수 딥RL/MARL을 이긴다 (CC3 우승 −1577 vs PPO −7617) |

**핵심 정식화**

| 요소 | 내용 |
|------|------|
| 3-에이전트 | Blue(방어)/Red(공격)/Green(정상 임무=가용성 기준) |

| 보상(혁신) | R_blue = Σ차단 − Σ침해 − λ·가용성손실 (가용성은 채점식에만 곱셈, 보상은 가중합) |
| 하이브리드 코어 | 휴리스틱(action-mask) + PPO + LLM(오프라인) + 안전가드(hard shield) |
