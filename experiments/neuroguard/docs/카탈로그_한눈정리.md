## 🧠 왜 휴리스틱이 RL/MARL을 이겼나 (CAGE 교훈 → ③ 설계 근거)
| 이유 | 설명 |
|------|------|
| 새 환경 + 백지 학습 | RL은 처음부터 다 배워야 함 ↔ 휴리스틱은 **전문가 지식 직접 인코딩**(학습 불요) |
| 무작위 토폴로지 | CC4는 호스트 수 매 에피소드 무작위 → MARL이 **무효 행동** 남발, 휴리스틱은 구조 규칙이라 무관 |
| MARL 병목 | 역할이 다른 에이전트인데 **파라미터 공유**가 정보 병목 → 협력 학습 비효율 |
| 부분관측·노이즈·희소보상 | RL이 약한 조건, 명시적 규칙이 강건 |
| 샘플·시간 제약 | RL은 막대한 학습 필요 → 제약 환경선 휴리스틱이 강한 baseline |

> 🔑 "항상 이긴다"가 아니라 **새·복잡·학습제약 환경**에서 강한 것. **최선은 하이브리드**(CC3 우승 = 7 PPO + 11 휴리스틱). → 우리 ③ **"휴리스틱 baseline + PPO 보강"** 의 직접 근거.
